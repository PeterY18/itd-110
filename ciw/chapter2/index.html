CSS versions

CSS1
basic structure of css
CSS2
Adds ability to import media types and work with tables
CSS3
Updates are made easier and more efficient. New functions were added that support color, backgrounds, borders and text effects

CSS Syntax

Selector
the html element that is being styled like body, h1, or p
Property
The trait that is being edited like background-color, color, or margin
Value
This is the argument for the property. For the property color, the value would be in hexdecimal format
CSS declaration
Everything between the curl braces including the semicolon, property, and value
CSS rule
The selector and all declarations combined is considered a CSS rule
Class
.class_name { declaration } is how class are written. class="class_name" is how to reference in html. if only element needs to referenced then h1.class_name {} is styelsheeet and h1 class="class_name is for referencing
id
#id_name and id="id_name" is for creating and referencing id

CSS Inheritance

  • A style sheet rule will override the default font color, which is black.
  • The internal stylesheet will override external stylesheets
  • Inline css overrides the other 2 stylesheets

Implement CSS

  • Declare inline css in html element
  • Use an internal stylesheet in head
  • link an html page to an external stylesheet
  • Font-family is the font that is used
  • font-size is takes pixel values to change text size. default is 16
  • color is the color of the text
  • link rel="stylesheet" type="text/css" href="css/habitat.css" is how to link an external stylesheet
  • rel defines the relationship between the two documents
  • type describes the document extension or MIME type
  • href specifies the path and location of the document
