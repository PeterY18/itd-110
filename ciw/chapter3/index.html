CSS selectors and properties

a[href$=".pdf"]
This selects all links with .pdf extension
a[href*="college"]
selects all links with the string college
a[href^="https"
selects all links that begin with https
element:[checked, disabled, enabled]
selects checked, disabled, or enabled elements
element:[first-of-type, last-of-type
selects element that is the first or last in its parent
element1~element2
Selects every instance of element2 that is preceded by an instance of element1. Selects every B that comes after element A

Background and border properties

background-clip
specifies painting area and determines if the background goes to the border
background-origin
positions background image relative to element's border, content, or padding
background-size
size of background, default auto. contain means to take up the whole screen. cover means to take up as much of the screen as possible while maintaining aspect ratio.
border-image
border is now an image
border-radius
defines curve on border
box-shadow
applied shadows the edge of elements
opacity
transparency values 0.0-1.0
Resize
defines if an element can be resized by the user
text-overflow
cuts or clips off text that extends beyond its container
text-shadow
applies shadows to text

font properties

text-shadow property
parameters are horizontal shadow, vertical shadow, blur distance, and color
word-wrap property
states if text breaks or wraps

More css properties

  • background, background-clip, background-origin, background-size, border-image, border-radius, box-shadow, opacity, resize, text-overflow, text-shadow
  • background-origin determines where background is relative to or where it starts from
  • background-clip determines if the background extends to edge of browser
  • background size determines pixel size or percentage size with the values like this background-size: 20% 20$
  • background-size: cover makes sure the whole webpage is covered
  • background-size: contain tries to cover as much space while keeping size rations intact
