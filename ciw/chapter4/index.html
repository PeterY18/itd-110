transform properties

transform
allows a 2D or 3D transformation of elements
transform-origin
change element's position
transform-style
determines if child will retain parent's position in 3D
perspective
Specifies the perspective from which a 3D child element is viewed by defining how far it is placed from view (in pixels)
perspective origin
specifies the bottom position of a 3D element
backface-visibility
defines if an element is viewable when rotated away from the viewer.

2D transforms

rotate(angle)
roates element in degrees
scale(x, y)
changes element size
matrix
combines all 2D transformations. 6 parameters which allow rotation, transform, scale, and skew
translate(x, y, z)
moves elements
skew(x-angle, y-angle)
turns or skews an element to a specified angle

3D transforms

3D transforms have rotate, scale, and translate. they have the z parameter because it is 3D.

matrix3D
Defines 3D transformation. uses 4x4 matrix
perspective(n)
Defines perspective for 3D elements. Changes z plane. smaller values bring the viewer closer and large values move the user farther away.

CSS3 Transitions

Transitions change the style (the css) of elements

Some properties are transition, transition-property, transition-duration, transition-timing-function, and transition-delay

CSS3 Animations

CSS3 animations replace plugins like adobe flash, apple quick time, and other javascript stuff

  • @keyframe rule defines animation and position
animation-iteration-count
defines how many times animation will play default is one
animation-direction
specifies whether to play animation in reverse or alternate cycles
animation-play-state
specifies if the animation is paused or running default is running.

CSS3 user interface properties

appearance
makes an element look like a standard user interface element, such as a button or window
box-sizing
forces specified elements to fit within parts of the box model
icon
styles an element with an icon equivalent
nav index
specifies tabbing order of elements
nav-[down, up, left, right]
spcifies where the cursor goes if the user hits the arrow keys
outline-offset
draws an outline beyond the border
resize
specifies if an element can be resized by the user or not. also specifies how much the user can resize the element
