HTML Terms

HTML
An abreviation for Hyper-Text Markup Language
HTML element
two angle brackets with a tag inside
HTML tag
the item that goes inside two angle brackets to form an eleemnt
angle bracket
the characters that surround a tag to form an element
HTML attribute
attributes are assigned a value using=
HTML comment
holds information about the HTML file for later use. Also used to hide errors. Comments are not seen by the browser or program.
HTML object element
Objects such as line breaks, horizontal ruling, and images are all object elements
HTML special character
Characters that are special in HTML so they need a sequence of other characters to be shown in HTML
