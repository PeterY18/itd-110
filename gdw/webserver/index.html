Web Server Project

Vocabulary

Internet
A global web of computers connected to servers to take in requests and send back information to the requester
TCP/IP
Transport Control Protocol/Internet Protocol are the standards to send and receive webpages
LAN
Land Area Network is similar to the internet but it is limited to a small local area
IP address
An address that is readable by the web browser that the computer sends when it wants to retrieve information. Often it is replaced with a DNS to make it easier for humans
HTTP
Hyper Text Transfer Protocol is the protocol for transferring HTML or hyper-text files
Web Server
Software that is designed to store website files and then handle requests of them
Web Client
A computer that makes a request of websites. This is often done by a web browser
localhost
The name of the local machine. Localhost connects to the same page as does the ip address of the machine.
hostname
A name given to a computer so that other computers can communicate with the machine
Avahi and Zeroconf
Zero configuration networking automatically sets up a computer network by setting up DHCP and other protocols. Avahi is a free software version of zeroconf
superuser
A user that is able to execute administrative commands with sudo or su
