Web Server Project

Vocabulary

Internet
A global web of computers connected to servers to take in requests and send back informaton to the requester
TCP/IP
Transport Control Protocol/Internet Protocol are the standards to send and recieve webpages
LAN
Land Area Network is similiar to the internet but it is limited to a small local area
IP address
An address that is readable by the webrowser that the computer sends when it wants to retrieve information. Often it is replaced with a DNS to make it easier for humans
HTTP
Hyper Text Transfer Protocol is the protocol for transferring HTML or hyper-text files
Web Server
Software that is designed to store website files and then handle requests of them
Web Client
A computer that makes a request of websites. This is often done by a web browser
